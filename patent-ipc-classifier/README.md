# Patent IPC Classifier

## Project Overview
A text classifier that predicts the IPC section (A–H) of a patent
based on its abstract, using sentence embeddings.

## Dataset
- Source: European Patent Office (EPO) via OPS API
- Size: ~4000 patents (500 per IPC section)
- Labels: 8 IPC sections (A through H)

## Models Trained
- Logistic Regression
- Support Vector Machine (SVM)
- Random Forest
